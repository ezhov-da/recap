reCAPTCHA v2 Checkbox

  • Отражается кнопка применить и счётчик попыток.
  • При нажатии 3 раза на счётчик (Counter) показывается капча.
  • Если её пройдешь, то отразится ответ капчи ниже.
  • Если продолжить нажимать, то капча скроется и при количестве кратно 3 снова отразиться сброшенная.
  • И при её прохождении будет уже другой ответ.
Какие кейсы пытался покрыть:
  • Управление отображение капчи по условию (получилось)
  • Получить callback при успешном прохождении капчи и выполнении логики (получилось)
  • Скрытие капчи и её reset для последующего использования (получилось)
